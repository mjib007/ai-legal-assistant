# 合約審核工作流程

## 🎯 審核目標
確保合約條款符合法律規定，保護當事人權益，降低法律風險。

## 📋 審核步驟

### 第一階段：基本資訊檢查
- [ ] 確認當事人資格
- [ ] 檢查合約標的物
- [ ] 驗證合約期限
- [ ] 確認權利義務條款

### 第二階段：風險評估
- [ ] 違約責任條款 ⚠️
- [ ] 損害賠償限制 🚫
- [ ] 管轄法院約定 ✅
- [ ] 適用法律選擇 ✅

### 第三階段：合規檢查
- [ ] 是否違反強制規定
- [ ] 是否符合消費者保護法
- [ ] 是否涉及不公平條款
- [ ] 是否需要主管機關核准

## 🚨 紅燈警示（立即停止）
- 違反法律強制規定
- 顯失公平條款
- 無效或得撤銷條款

## ⚠️ 黃燈注意（需要修改）
- 條款不夠明確
- 風險分配不當
- 缺少必要約定

## ✅ 綠燈通過（可以接受）
- 條款明確合理
- 風險分配適當
- 符合法律規定

## 📝 建議修改範本
**違約金條款調整：**
```
建議修改為：「違約金不得超過契約總金額的20%」
```

**管轄權約定：**
```
建議加入：「因本契約發生之爭議，同意以台灣台北地方法院為第一審管轄法院」
```
