# AI 法律助手

這是一個簡單的 AI 法律諮詢助手範例，展示如何整合 AI 技術與法律服務。

## 功能特色
- 法律條文查詢
- 簡單法律問題解答
- 合約範本產生器

## 如何使用
1. Clone 這個專案
2. 用瀏覽器開啟 src/index.html
3. 開始使用法律助手

## 專案架構
```
ai-legal-assistant/
├── README.md
├── src/
│   ├── index.html
│   ├── script.js
│   └── style.css
└── examples/
    └── sample-queries.md
```
